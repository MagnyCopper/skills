# Western Europe — 英国 / 爱尔兰 / 法国 / 比利时 / 荷兰

## 英国 (United Kingdom / GB)

```
# C-GB-1: UK ccTLD
"<company_name>" site:.uk OR site:.co.uk

# C-GB-2: Companies House reference
"<company_name>" "Companies House" site:.gov.uk
```

For UK companies, cross-checking with Companies House (`https://find-and-update.company-information.service.gov.uk/`) is recommended.

Preferred engines: Google (primary), Bing (secondary).

## 法国 (France / FR)

```
# C-FR-1: French ccTLD
"<company_name>" site:.fr

# C-FR-2: Mentions légales (legally required)
"<company_name>" "mentions légales"

# C-FR-3: French name if discoverable
<company_name_french> site officiel
```

French company sites are required to have "mentions légales" (legal mentions). Finding this page with matching company details is a strong signal.

Preferred engines: Google (primary), Bing (secondary).
