# Americas — 美国 / 加拿大 / 巴西 / 阿根廷 / 智利 / 哥伦比亚 / 墨西哥

## 美国 (United States / US)

```
# C-US-1: US companies typically use .com
"<company_name>" site:.com

# C-US-2: SEC EDGAR for public companies
"<company_name>" site:sec.gov EDGAR

# C-US-3: State registry
"<company_name>" "Secretary of State" <state>
```

For publicly traded US companies, SEC EDGAR filings often contain the official website URL.

Preferred engines: Google (primary), Bing (secondary).
