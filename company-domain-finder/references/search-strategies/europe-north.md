# Northern Europe — 德国 / 瑞士 / 瑞典 / 挪威 / 丹麦 / 芬兰

## 德国 (Germany / DE)

```
# C-DE-1: Impressum search (legally required for German companies)
"<company_name>" Impressum

# C-DE-2: German ccTLD
"<company_name>" site:.de

# C-DE-3: Combined
"<company_name>" site:.de OR site:.com "Impressum"
```

German company sites almost always have an "Impressum" (legal notice) page. Finding an Impressum that matches the company name is a very strong verification signal.

Preferred engines: Google (primary), Bing (secondary).
