## Usage Notes

- Registries are for **cross-validation only**, not primary discovery. Use them when search results are ambiguous or when you need to confirm that a domain truly belongs to the registered company.
- Not all registries expose company website URLs. Many only provide registered addresses and officer information.
- Some registries require local language input or specific identifiers (e.g., CNPJ for Brazil, unified social credit code for China).
- When a registry shows a registered address that matches the address on a candidate website, this is a strong verification signal.
